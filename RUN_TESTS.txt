================================================================================
  HOW TO RUN THE TESTS (After Installing Icarus Verilog)
================================================================================

IMPORTANT: You must run these commands from Windows Command Prompt (cmd.exe),
           NOT from Git Bash, because Icarus was added to the Windows PATH.

================================================================================
STEP 1: Open Windows Command Prompt
================================================================================

Press:  Windows Key + R
Type:   cmd
Press:  Enter

================================================================================
STEP 2: Navigate to Project Directory
================================================================================

cd r:\fpga\fpga-power-test-accel

================================================================================
STEP 3: Verify Icarus is Installed
================================================================================

iverilog -v

You should see:
  Icarus Verilog version 12.0 (stable) ...

If you see "command not found", you need to:
  1. Restart your computer (to refresh PATH)
  2. Or reinstall Icarus and check "Add to PATH"

================================================================================
STEP 4: Run All Tests
================================================================================

scripts\run_iverilog_tests.bat

You should see:
  =======================================================
    Accelerator Infrastructure Test Suite (Icarus)
  =======================================================

  Running: Memory Controller Test
  -------------------------------------------------------
    PASSED

  Running: Address Generator Test
  -------------------------------------------------------
    PASSED

  [... more tests ...]

  =======================================================
    Test Summary
  =======================================================
  Passed: 5
  Failed: 0
  Total:  5

  All tests passed!

================================================================================
STEP 5: View Detailed Results
================================================================================

Test logs are saved in: build\test_results\

To view:
  dir build\test_results
  type build\test_results\tb_memory_controller_sim.log

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "iverilog is not recognized"
Solution:
  1. Close ALL command prompts and terminals
  2. Restart computer
  3. Open NEW command prompt
  4. Try again

Problem: Tests fail to compile
Solution:
  Check: build\test_results\*_compile.log for error details

Problem: Tests compile but fail simulation
Solution:
  Check: build\test_results\*_sim.log for error details

================================================================================
ALTERNATIVE: Run Single Test Manually
================================================================================

To test just the memory controller:

iverilog -g2012 -o build\test.vvp ^
    common\rtl\accel_pkg.sv ^
    common\rtl\bram_bank.sv ^
    common\rtl\memory_controller.sv ^
    common\tb\tb_memory_controller.sv

vvp build\test.vvp
